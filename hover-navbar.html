Hover-nav

CDN link

To use the hover-nav component, include the following CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/hovernav/hovernav.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/hovernav/hovernav.js" defer> </script>

Hover nav provides the classic website navigation with a sliding effect.

Attributes

Attribute Value
pages Name of the pages (separate with commas)
links Links for the page (URL or .html file)
imageicons Icons for your menu (separate with commas)
menubtncolor Color of the menu button
menubtnborder CSS border property
menustyle Menu color, hover color
iconstyle Icon background color, hover color
fontstyle Font color, hover color
iconborder CSS border property
openicon Image (e.g., hamburgericon)

Customize the hover-nav's appearance and behavior using these attributes.

<hover-nav pages="home,about,contact" links="link1.html,link2.html,link3.html" imageicons="../home.png,../about.png,../contact.png" menubtncolor="orange" menubtnborder="10px solid black" menustyle="orange,black" iconstyle="white,orange" fontstyle="black,white" iconborder="3px solid black"> </hover-nav>
