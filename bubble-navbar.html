Bubble-nav

CDN link

To use the bubble-nav component, include the following CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/bubblenav/bubblenav.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/bubblenav/bubblenav.js" defer> </script>

A circular menu that pops up from the corner, allowing you to create interactive and visually appealing navigation.

Note: bubble-nav can handle only up to 4 menu items.

Attributes

Attribute Value
color Color of the bubbles
hovercolor Color on hover
bubbletext-color Font color
bubbleborder Changes the border color of the bubble
font Font family of your choice
pages Names of the pages (separate with commas)
links Links for the pages (URL or .html file)
bubble1-x The X position of bubble 1 in px
bubble1-y The Y position of bubble 1 in px
bubble2-x The X position of bubble 2 in px
bubble2-y The Y position of bubble 2 in px
bubble3-x The X position of bubble 3 in px
bubble3-y The Y position of bubble 3 in px
bubble4-x The X position of bubble 4 in px
bubble4-y The Y position of bubble 4 in px
menucolor Color of the menu button in the corner
menutext-color Font color of the menu button
menuborder Use CSS border (e.g., 2px solid red)

By using these attributes, you can customize the bubble menu's appearance and functionality to suit your website's needs.

<bubble-nav pages="contact,about,product,new" links="contact.html,about.html" color="orange" hovercolor="grey" bubbletext-color="black" bubbleborder="red" font="arial"> </bubble-nav>
