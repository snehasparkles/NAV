Simple-nav

CDN link

To use the simple-nav component, include the following CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/simplenav/simplenav.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/simplenav/simplenav.js" defer> </script>

Simple nav provides a minimalistic navigation bar without any advanced effects.

Attributes

Attribute Value
msg Message to be displayed on top of the navbar
msgcolor Background color, font color for the message
navbarcolor Bar background color, font color, menu hover color, font hover color
pages Names of the pages (separate with commas)
links Links for the pages (URL or .html file)
logo Add your brand image
logodim Dimensions: Height and Width in pixels (px)
icons Icons for your social media (separate with commas)
iconlinks Links for the icons (URL)
iconhovercolor Hover color on the icons

Customize the simple-nav's appearance and behavior using these attributes.

<simple-nav msg="Hello World!" msgcolor=",white" navbarcolor="green,white,lightgreen,black" pages = "Home,Blog,Service,Contact,new,works" links ="../index.html,../about.html,link3.html,link4.html" logo = "images/logo.jpeg" logodim="30px,40px" icons="images/gg.png,images/fb.png,images/ig.png,images/yt.png,images/wt.png" iconlinks="https://www.google.com,,link3.html,https://www.youtube.com,link4.html" iconhovercolor="lightgreen"> </simple-nav>
