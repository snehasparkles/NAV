Getting Started

How to Use NAV ?

Include the CDN Links: Start by including the respective CSS and JavaScript CDN links in the section of your HTML document.

Add NAV Tags: Use the NAV tags in your HTML to define the structure and style of your navbar.

Customize with Attributes: Each NAV tag comes with a set of attributes that allow you to customize the navbar's appearance and behavior.

Example:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/basic-nav/style.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/basic-nav/script.js" defer> </script> <basic-nav logo="path/to/logo.png" brand="My Brand" pages="Home,About,Contact" links="home.html,about.html,contact.html"> </basic-nav>

Each navigation tag within the framework comes with its own unique CDN link and specific attributes tailored to the type of navigation bar you wish to create.

For example, the <basic-nav> tag uses a distinct CDN link that loads the necessary CSS and JavaScript files to render a basic navigation bar. This tag allows you to customize the logo, brand name, and links through its attributes. Similarly, other tags like <hover-nav>, <slide-nav>, and <float-nav> have their own CDN links and attribute sets, enabling developers to easily implement different navigation styles with minimal effort.

By including the appropriate CDN links and using the correct attributes, you can quickly build a variety of navigation bars that enhance the user experience on your website or web application.
