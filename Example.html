Basic-Nav

This tag is used to create a simple, customizable, and responsive navigation bar with branding, menu links, and a mobile-friendly toggle icon.

Bubble-Nav

This tag is ideal for playful or modern websites, featuring an animated navigation menu with bubble-like buttons for enhanced engagement.

Float-Nav

Perfect for floating, customizable navigation menus, ideal for portfolios, blogs, or sleek, modern web apps.

HalfSlide-Nav

This tag is Great for modern websites needing a space-saving menu with a half-slide effect, suitable for creative portfolios or trendy e-commerce sites.

Hover-Nav

This tag creates a dynamic navigation menu with hover effects and icons, ideal for creative, interactive, or design-focused websites.

Pop-Nav

This tag provides a bold, pop-up navigation menu, perfect for vibrant fashion sites, creative portfolios, or impactful web designs.

Simple-Nav

This tag creates a simple, customizable navigation bar with social icons and a welcome message, ideal for blogs, small businesses, and personal portfolios.

Slide-Nav

This tag creates a sleek, left-sliding navigation menu, ideal for modern, space-efficient websites and apps with interactive design.
