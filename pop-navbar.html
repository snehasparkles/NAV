Pop-nav

CDN link

To use the pop-nav component, include the following CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/popnav/popnav.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/popnav/popnav.js" defer> </script>

Pop nav provides a navigation bar with a popping effect.

Attributes

Attribute Value
openicon Image that opens menu (e.g., hamburgericon)
closeicon Image that closes menu (e.g., closeicon)
btncolor Color of the navbar button
btnborder CSS border property for the button
menustyle Bar background color, menu color, hover color, active color
fontstyle Font color, hover color
pages Names of the pages (separate with commas)
links Links for the pages (URL or .html file)

Customize the pop-nav's appearance and behavior using these attributes.

<pop-nav openicon="images/hamburger.png" closeicon = "images/exit.png" btncolor="wheat" btnborder = "5px solid lightgrey" menustyle = "wheat,black,white," fontstyle="white,black" pages="home,about,contact,new product" links="link1.html,link2.html,link3.html"> </pop-nav>
