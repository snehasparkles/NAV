Halfslide-nav

CDN link

To use the halfslide-nav component, include the following CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/halfslidenav/halfslide.css"> <script src="https://cdn.jsdelivr.net/gh/MANIKANDANMLT/nav@main/halfslidenav/halfslide.js" defer> </script>

Halfslide nav provides the classic website navigation with a sliding effect.

Attributes

Attribute Value
slideicon Image that opens menu (e.g., hamburgericon)
icon Add your brand image
logodim Dimension Height and Width in px
barcolor Color of the navbar
slidecolor Color of the slide
fontstyle Font color, hover color
pages Name of the pages (separate with commas)
links Links for the page (URL or .html file)
right Slides from right side if right="true"

Customize the halfslide-nav's appearance and behavior using these attributes.

<halfslide-nav slideicon="images/hamburger.png" icon="../nav.jfif" logodim="40px,50px" barcolor="#FABC3F" slidecolor="grey" fontstyle="white,black" pages="home,about,contact,services,blog" links="link1.html,link2.html,link3.html" right=""> </halfslide-nav>
